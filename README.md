## 3. 判定标准（核心）
### 3.1 题目一：医生是否偏离指南（单选）

#### (1) 非高血压用药建议

**定义**：医生的建议完全不涉及高血压相关药物的调整。
**适用情况**：
* 仅建议生活方式改善（如低盐饮食、运动）或建议去医院检查，未开具任何处方。
* 仅开具了与心血管完全无关的药物（如感冒药、安眠药、钙片等）。

#### (2) 遵循指南

**定义**：医生的用药建议完全符合高血压治疗规范，是标准推荐方案。
**适用情况**：
* 方案与《指南》推荐的一线用药方案一致。
* 药物选择、剂量、频次均准确无误。
*  **注意**：如果医生**只提了药名，没写具体剂量/频次**，**可判为“遵循指南”**，不要因为没写剂量判错。

#### (3) 偏离但可接受

**定义**：医生的用药建议虽偏离指南推荐，但**没有安全风险**。

#### (4) 偏离且不可接受

**定义**：医生的建议触犯了安全红线，存在明显的医疗风险或错误。
**适用情况**：
* 涉及禁忌症、重复用药、严重超量/少量、药物相互作用等原则性错误。

---

### 3.2 题目二：偏离指南的维度（多选）

**触发条件**：如果在题目一中选择了 **(3) 偏离但可接受** 或 **(4) 偏离且不可接受**，则本题为**必填项**。

#### [1] 剂量/用法错误

**判定标准**：医生**明确写出**的剂量或频次违反指南或说明书。
* *例：氨氯地平建议每日 500mg（极量通常为10mg）。*
* *例：指南建议每日一次用药，但医生推荐每日三次。*
* **注意**：再次强调，**未提及剂量 ≠ 剂量错误**。

#### [2] 禁忌症

**判定标准**：药物与患者的生理状况、病史或过敏史冲突。
* *例： **妊娠/备孕妇女**：禁用 **普利类(ACEI)** 或 **沙坦类(ARB)**。*
* *例： **哮喘患者**：禁用 **普萘洛尔** (非选择性β阻滞剂)。*

#### [3] 重复用药

**判定标准**：同时开具两种或以上同类或同机制的药物。
*  *例：**普利类 (XX普利) + 沙坦类 (XX沙坦)**：这是最典型的重复用药，严禁合用。*
*  *例： 同一种成分开了两个不同厂家的药。*

#### [4] 药物相互作用

**判定标准**：两种不同机制的药物合用，导致药效降低或毒性增加。
* *例：非二氢吡啶类CCB（维拉帕米/地尔硫卓）与 β受体阻滞剂（倍他乐克）合用，可能导致严重心动过缓。*

## 4、常见问题 (FAQ)
* **Q: 医生建议“监测血压”，算什么？**
    * A: 如果没开药，算【非高血压用药指导】。
* **Q: 患者说自己怀孕了，但医生没接话，直接开了“厄贝沙坦”，怎么判？**
    * A: 判【禁忌症】。结果就是给孕妇开了禁忌药，必须拦截。
* **Q: 医生说“原来的药继续吃，再加个XX药”，原来的药也要审核吗？**
    * A: **要审核**。医生的建议构成了“新处方+旧处方”的联合方案，你需要评估这个**最终组合**是否遵循指南（比如新加的药是否和旧药重复）。
